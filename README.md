# 2S2H ROM Converter
If you want to play 2ship2harkinian, you'll need to provide a _Majora's Mask_ ROM. Unfortunately, it only works with NTSC-U versions of the game.

If your dumped ROM is the wrong version, you can use these patches to convert it to the correct version.

1. Download the `patches.7z` file from this repository.
2. Extract the patch that matches the version of your ROM.
3. Use [Rom Patcher JS](https://www.marcrobledo.com/RomPatcher.js/) or your preferred ROM patcher to apply the patch to your ROM.
4. You should now have an NTSC-U ROM that can be used with 2s2h.
